## What this is

Quarterly community-voted priorities (3 push goals + 2–3 don't-drop-the-ball items), plus fortnightly meetings, progress updates, and bounty orchestration to move them. The Keeper facilitates; the community builds. See [PROPOSAL.md](./PROPOSAL.md) for full scope, the payment gate, and the locked pro-ration.

This README tracks live status and links and will change over time. The binding contract is the tagged `PROPOSAL.md@v1-proposal`, which does not.
